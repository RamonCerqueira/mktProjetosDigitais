# Portal de Projetos Digitais (Next.js)

Este repositório agora é **100% frontend/fullstack em Next.js**, sem backend Java.

## O que mudou
- Removido backend Java do projeto.
- Home remodelada para um visual próximo ao layout de referência (hero, busca, métricas e cards em destaque).
- Experiência de "portal vivo": indicadores atualizados em tempo real no topo da página.

## Stack
- Next.js App Router
- TypeScript
- TailwindCSS
- API routes em `frontend/src/app/api/v1/*`

## Rodar localmente
```bash
cd frontend
npm install
npm run dev
```

A aplicação abre em `http://localhost:3000`.

## Ambiente (opcional)
Copie o exemplo e ajuste se necessário:
```bash
cp .env.example .env
```
